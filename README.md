## 数据结构与算法

#### 1. 两者的关系

> - 数据结构就是指一组数据的存储结构、组织方式。如果不在它的基础上操作、构建算法，孤立存在的数据结构就是没用的。
> - 算法就是操作数据的一组方法。
> - 数据结构和算法是相辅相成的。数据结构是为算法服务的，算法要作用在特定的数据结构之上。
