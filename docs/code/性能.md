# 性能

> 数据结构或者算法的性能，衡量标准：时间复杂度和空间复杂度。

### 1. 时间复杂度

算法运行时间。

时间复杂度常用**大O符号（Big-O）**表示，表示当数据结构的元素增加的时候，算法规模或者性能在最坏场景下有多好。

#### 常见时间复杂度

| Big-O | 复杂度量级 |
| :--- | :---: |
| O(1) | 常数阶 |
| O(log n) | 对数阶 |
| O(n) | 线性阶 |
| O(n log n) | n log n 阶 |
| O(n^2) | 平方阶 |
| O(n^3) | 立方阶 |
| O(2^n) | 指数阶 |
| O(n!) | 阶乘阶 |

### 2. 空间复杂度

算法运行所需的内存空间。
