# 链表（Linked List）

#### 1. 特征

- 链式存储
- 新增、删除快

#### 2. 注意点

- 边界条件（头尾节点 NULL 值）处理
- 形成环
- 内存泄漏

#### 3. 技巧

1. 头节点（dummy）：创建一个无意义的头节点，解决链表中头指针带来的一堆判断或者其他逻辑。
2. 多用个临时变量。如果多使用几个变量能够简化不少代码逻辑（主要是条件判断），使代码更加美观和通俗易懂，那么千万不要吝啬那点内存。
3. 头插法
4. 双指针
5. 倒序处理和递归
6. 可以用双数组（索引数组）模拟一个链表。
